## Perguntas

### Quiz 1
> What type of software is Git?

- Controlador de versões

> What is one feature that is not a difference between Git and GitHub?

- Github contêm um terminal especial web para interagir com o seu repositório

> What is the best terminal to use across all operating systems?

- Bash

### Quiz 2
> What is the correct command to go back one level?

- Comando `cd ..`

> How would I create a folder called dir in the terminal?

- Comando `mkdir dir`

> What does ~ stand for in the terminal?

- Aponta para o caminho padrão do sistema

> What is the correct command to change directories?

- Comando `cd /path`

> How are directories removed in the terminal?

- Comando `rm -rf nome`.

### Quiz 3
> What is the HEAD?

- O commit selecionado atualmente

> True or False: You can commit as soon as you make changes.

- Falso

> What does the command `git add .` do?

- Aceita todas as alterações feitas no repositório para o commit atual

> True or False: `git rm -f` removes the file from the working directory

- Verdadeiro

> What is the command used to create a new repository?

- Comando `git init`

- Falso

- Uma corrente de commits que não afeta commits de outras branches

> True or False: You can have multiple HEAD pointers when having more than 1 branch.

- Falso

- Um ciclo de **push** e **pull**
